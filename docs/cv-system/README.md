# CV Generation System - Gabriel Cárcamo

**Created:** 2025-11-24
**Purpose:** Centralized system for generating tailored CVs for specific job applications
**Status:** Active

## System Overview

This CV System consolidates all your professional experience, achievements, and strategic positioning to enable rapid generation of tailored CVs for specific job applications.

## Components

1. **[master-cv.md](master-cv.md)** - Complete professional profile (source of truth)
2. **[foda-analysis.md](foda-analysis.md)** - Strategic positioning (strengths, weaknesses, opportunities, threats)
3. **[target-roles.md](target-roles.md)** - Role profiles and positioning strategies
4. **[success-stories.md](success-stories.md)** - STAR format achievement library
5. **[skills-matrix.md](skills-matrix.md)** - Keywords, technologies, and ATS optimization
6. **[tailoring-rules.md](tailoring-rules.md)** - Guidelines for customizing CVs per role

## Workflow

```
1. Identify target role → Consult target-roles.md
2. Select relevant experiences → Reference master-cv.md
3. Choose appropriate stories → Use success-stories.md
4. Apply positioning strategy → Follow foda-analysis.md
5. Optimize for ATS → Check skills-matrix.md
6. Apply tailoring rules → Use tailoring-rules.md
7. Generate final CV → Compile and format
```
