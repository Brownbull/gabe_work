## Role-Specific CV Tailoring Quick Reference

### For Each Application:

1. **Identify Role Type** (use categories above)
2. **Adjust Professional Summary** (use "Lead With" section)
3. **Select Relevant Experience** (emphasize aligned projects)
4. **Choose Achievement Bullets** (use sample bullets or adapt)
5. **Update Skills Section** (highlight relevant tech stack)
6. **Address Gaps Proactively** (if applicable)

### Example: Applying for "Senior Python Engineer at Xmartlabs"

Based on Xmartlabs research:
- **Company Focus:** Mobile, web, ML applications
- **Stack:** Python, ML/AI, AWS, mobile-friendly backends
- **Culture:** Open source, code quality, mentorship

**Tailoring Strategy:**
- Lead with Python depth (13+ years)
- Emphasize open-source mindset (GitHub projects)
- Highlight ML projects (align with their AI/ML focus)
- Mention code quality and review practices
- Show mentorship experience
- Include Flask production experience
- Mention ML frameworks (align with their TensorFlow/PyTorch)

**De-emphasize:**
- Mainframe tech (not relevant)
- Client-facing work (less important for IC role)
- Management aspirations (not requested)
