# Quick Start

Goals
- Capture ideas fast, track mood, and ship posts consistently.

1) New post
- Copy Posts/_template to Posts/YYYY-MM-DD-title-kebab
- Example: Posts/2025-08-21-llm-offline-eval
- Open post.md and update front matter (title, slug, dates, pillar, hashtags)

2) Write flow
- Fill 00-brief.md (Outcome, Thesis, Audience, Why now, Key points)
- Draft 01-outline.md (hook, H2s, bullets)
- Write 02-draft.md quickly (allow TODOs)
- Polish in 03-polish.md; run checklist.md
- Prepare 04-social.md (hooks, replies, hashtags)
- Put any links/quotes in references.md and research.md

3) Assets and export
- Add images/diagrams to assets/
- Save final copy to exports/final.md

4) Plan area
- Add new ideas to Plan/10-backlog/backlog.md and Plan/10-backlog/ideas/
- Log good examples in Plan/30-research/swipe-file.md
- Track metrics in Plan/50-metrics/
- Update calendar in Plan/20-calendar/YYYY-MM.md

5) Mood tracking
- Append a new line to Plan/60-mood/mood-log.csv (date,mood,energy,focus,block,notes)
- Optional: Journal in Plan/60-mood/mood-journal.md

Naming
- Folder: YYYY-MM-DD-title-kebab
- Status in post.md: draft | outline | writing | polish | scheduled | published

Cadence
- 1-2 long-form posts/week
- Derive 2-3 shorts per long post

Tip
- Use Plan/00-framework/prompts.md to ideate, outline, tighten, and prepare CTAs.
